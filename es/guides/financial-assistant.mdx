---
title: Agente Financiero
---

En la esquina inferior derecha, el sistema muestra un botón de acceso al **agente financiero con inteligencia artificial**, el cual permite obtener asistencia automatizada sobre información financiera.

Al seleccionar este botón, se abre una ventana de conversación donde usted puede interactuar con el asistente virtual.

![title](/images/cedi-agente-1.png)
---

### Funcionalidades del agente financiero

El agente financiero permite a usted realizar consultas relacionadas con la información financiera del sistema.


Entre las consultas que puede realizar se encuentran:

- Consultar el flujo financiero mensual  
- Revisar pagos realizados a proveedores  
- Obtener un resumen de ingresos y egresos  
- Identificar movimientos pendientes  
- Resolver dudas relacionadas con su información financiera  

Adicionalmente, el sistema permite adjuntar archivos para análisis financiero.

Usted puede:

- Subir archivos en formato **Excel** o **CSV**  
- Solicitar análisis o interpretación de la información  
- Enviar instrucciones mediante el campo de texto  

Para enviar una consulta, usted debe escribir su solicitud en el campo correspondiente y seleccionar el botón de envío.

---

### Uso del agente financiero para análisis de información

El agente financiero puede utilizar los archivos cargados por usted para generar análisis automatizados y responder preguntas relacionadas con:

- Estados financieros  
- Movimientos contables  
- Pagos y cobros  
- Tendencias financieras  
- Información fiscal o administrativa  

Esta funcionalidad permite agilizar el análisis de información y facilitar la toma de decisiones financieras dentro del sistema.
