## Módulo Transferencias

Usted accede al módulo **"Transferencias"** desde el menú lateral del sistema. Al ingresar, el sistema muestra la pantalla principal, donde se presentan los elementos clave necesarios para la gestión de sus operaciones.

En esta vista, el sistema muestra su **CLABE**, el **saldo disponible**, así como los botones de acción principales: **"Nueva transferencia"**, **"Transferencia internacional"** y **"Descargar estado de cuenta"**. Esta información le permite tener un control inmediato sobre sus recursos y opciones disponibles.

---

Usted puede visualizar su CLABE en cualquier momento, así como consultar el saldo disponible actualizado. Para iniciar una transferencia nacional, debe seleccionar la opción **"Nueva transferencia"**.

---

### Movimientos recientes

El sistema presenta la sección **"Movimientos Recientes"**, donde se muestra el historial de transacciones realizadas. En este apartado, usted puede buscar y filtrar la información según distintos criterios, como nombre, fecha, tipo o estatus.

Una vez aplicados los filtros, el sistema muestra la lista de transferencias recientes, permitiéndole consultar el detalle de cada operación de manera clara y organizada.
