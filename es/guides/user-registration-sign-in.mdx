---
title: Inicio de sesión
description: Proceso para la creación, validación y activación de usuarios en el sistema CEDI
---


El usuario ingresa a la página principal del sistema y selecciona la opción **"Crear cuenta"** para iniciar el proceso de registro. A continuación, introduce la información solicitada en los campos correspondientes, incluyendo su correo electrónico, nombre, apellido y contraseña.

Una vez capturada la contraseña, el sistema evalúa automáticamente si cumple con los requisitos de seguridad establecidos. Para ser válida, la contraseña debe tener al menos 8 caracteres, incluir al menos una letra mayúscula, un número y un carácter especial (como $%&_). Si la contraseña cumple con todos los criterios, el sistema muestra una confirmación visual indicando que es válida. En caso contrario, se informa al usuario cuáles requisitos no se cumplen, permitiéndole corregirla.

Posteriormente, el usuario acepta los **términos y condiciones** y selecciona la opción **"Crear cuenta"** para continuar.

![title](/imagesu/cedi-crearcuenta.png)

El sistema envía un código de confirmación al correo electrónico proporcionado. El usuario debe ingresar este código en el campo correspondiente y hacer clic en **"Verificar código"**. Si el código es correcto, el sistema le permite continuar con el proceso de registro. En caso de que el código sea inválido, se notifica al usuario y se le brinda la opción de solicitar el reenvío del mismo.

![title](/imagesu/cedi-codigo.png)

Una vez verificada la cuenta, el usuario es redirigido a la pantalla de inicio de sesión, donde debe ingresar el correo electrónico registrado y su contraseña. El sistema valida las credenciales proporcionadas; si son incorrectas, se muestra un mensaje indicando el error. Si los datos son válidos, el usuario puede acceder y continuar dentro del sistema.

![title](/imagesu/cedi-iniciarsesion.png)
