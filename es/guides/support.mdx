## Módulo Soporte

Usted accede al módulo **"Soporte"** desde el menú lateral del sistema. Al ingresar, el sistema muestra la ventana principal del chat **Soporte CEDI**, diseñada para brindarle asistencia directa.

En esta pantalla, el sistema presenta el mensaje **"Chatea con Soporte CEDI"** acompañado de la indicación **"¿Cómo puedo ayudarle?"**, invitándolo a iniciar una conversación. Asimismo, se muestra un campo de texto donde usted puede escribir mensajes relacionados con dudas, solicitudes o problemas dentro de la plataforma.

Redacte su petición en el campo correspondiente y seleccione el botón de **enviar**. El sistema recibe el mensaje, lo procesa y responde mediante un asistente virtual especializado en resolver consultas relacionadas con el funcionamiento del sistema.

---

### Nueva conversación

Usted tiene la opción de iniciar una nueva conversación en cualquier momento seleccionando el botón correspondiente. Al hacerlo, el sistema genera una nueva sesión de chat, limpia el historial visible de la conversación anterior y habilita un nuevo espacio para comenzar una interacción distinta.

Esta funcionalidad permite separar consultas diferentes y mantener una organización clara dentro del módulo de soporte.

---

### Conversaciones anteriores

El sistema también permite consultar el historial de conversaciones previas. Para ello, usted puede seleccionar el menú de conversaciones, donde se muestra una lista de interacciones recientes.

Puede elegir cualquiera de las conversaciones anteriores, y el sistema cargará automáticamente el historial completo de la conversación seleccionada. A partir de ese momento, usted puede continuar enviando mensajes dentro de esa misma sesión.

El sistema mantiene el registro de cada interacción realizada y permite cambiar entre conversaciones sin perder la información previamente almacenada, garantizando continuidad y seguimiento adecuado de las solicitudes de soporte.
