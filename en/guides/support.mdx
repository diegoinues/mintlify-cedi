## Support Module

You access the **"Support"** module from the system side menu. Upon entering, the system displays the main window of the **CEDI Support** chat, designed to provide you with direct assistance.

On this screen, the system presents the message **"Chat with CEDI Support"** accompanied by the prompt **"How can I help you?"**, inviting you to start a conversation. Likewise, a text field is displayed where you can write messages related to questions, requests, or problems within the platform.

Write your request in the corresponding field and select the **send** button. The system receives the message, processes it, and responds through a virtual assistant specialized in resolving queries related to the operation of the system.

---

### New conversation

You have the option to start a new conversation at any time by selecting the corresponding button. By doing so, the system generates a new chat session, clears the visible history of the previous conversation, and enables a new space to begin a different interaction.

This functionality allows separating different queries and maintaining clear organization within the support module.

---

### Previous conversations

The system also allows consulting the history of previous conversations. To do this, you can select the conversations menu, where a list of recent interactions is displayed.

You can choose any of the previous conversations, and the system will automatically load the complete history of the selected conversation. From that moment on, you can continue sending messages within that same session.

The system maintains a record of each interaction performed and allows switching between conversations without losing previously stored information, guaranteeing continuity and proper follow-up of support requests.
