title: National
---

To initiate a transfer, you must access the **"Transfers"** module from the system side menu. On the main screen, the system displays the **"Recent Movements"** section, where you can review the history of completed operations.

From this screen, you can select the **"New transfer"** option to begin a new process.

---

### Document selection (optional)

When starting a new transfer, the system displays the **"New Transfer"** screen, where you are asked to indicate whether you wish to attach documents related to the operation.

You can upload any of the following files:

- Invoice  
- Bank statement  

![title](/images/cedi-transferencia-nacional-1.png)

The system allows you to attach these documents to facilitate payment identification or later reconciliation. However, you can also continue without uploading documents if you wish.

If you select documents, the system displays a side window called **"Invoices to transfer"**, where the available invoices are listed for selection.


You can:

- Select one or multiple invoices  
- Upload new documents  
- Continue without selecting documents  

Once the selection has been made, you must select the **"Use in transfer"** option to continue with the process.

![title](/images/cedi-transferencia-nacional-2.png)
![title](/images/cedi-transferencia-nacional-3.png)
---

### Beneficiary information

Subsequently, the system displays the **"New beneficiary"** form, where you must enter the information corresponding to the person or company that will receive the transfer.

The requested data includes:

- CLABE or account number  
- Bank  
- Beneficiary name  
- RFC or CURP  
- Email address  
- Beneficiary alias (optional)  

The system automatically validates the entered information to ensure that the data is correct and compatible with the operation.

Once the data has been completed, you must select the **"Continue"** option to proceed to the next step.

![title](/images/cedi-transferencia-nacional-4.png)
---

### Transfer amount and details

On the next screen, the system displays the **"Amount"** form, where you must enter the financial information of the operation.

The requested fields include:

- Amount to transfer  
- Payment concept  
- Reference  
- Description or reason for the transfer  

The system automatically calculates the total amount of the operation, including the corresponding fee, and displays the final amount to be debited from your available balance.

You must carefully review the information before continuing.

Once the information has been validated, you must select the **"Send transfer"** option to proceed with the operation.

![title](/images/cedi-transferencia-nacional-5.png)
---

### Security verification

Before completing the transfer, the system displays the **"Security verification"** window, where you are asked to enter an authentication code to confirm the operation.

This code corresponds to the previously configured security mechanism, such as two-factor authentication (**2FA**).

You must enter the code in the corresponding field and select the **"Verify"** option.

---

### Operation confirmation

Once the security code has been successfully validated, the system processes the transfer.

At the end of the process, the system:

- Confirms that the transfer was successfully completed  
- Updates the available balance  
- Records the movement in the **"Recent Movements"** section  

This allows you to maintain clear and up-to-date control of all operations performed within the system.
