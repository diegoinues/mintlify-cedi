title: Financial Agent
---

In the lower right corner, the system displays an access button to the **financial agent with artificial intelligence**, which allows you to obtain automated assistance regarding financial information.

By selecting this button, a conversation window opens where you can interact with the virtual assistant.

![title](/images/cedi-agente-1.png)

---

### Financial agent functionalities

The financial agent allows you to perform queries related to the financial information of the system.

Among the queries you can perform are:

- Check the monthly financial flow  
- Review payments made to suppliers  
- Obtain a summary of income and expenses  
- Identify pending movements  
- Resolve questions related to your financial information  

Additionally, the system allows you to attach files for financial analysis.

You can:

- Upload files in **Excel** or **CSV** format  
- Request analysis or interpretation of the information  
- Send instructions through the text field  

To send a query, you must write your request in the corresponding field and select the send button.

---

### Use of the financial agent for information analysis

The financial agent can use the files uploaded by you to generate automated analyses and answer questions related to:

- Financial statements  
- Accounting movements  
- Payments and collections  
- Financial trends  
- Tax or administrative information  

This functionality allows you to speed up information analysis and facilitate financial decision-making within the system.
