---
title: Sign-in
description: Process for creating, validating, and activating users in the CEDI system
---

The user opens the system home page and selects **"Create account"** to start registration. They enter the requested information in the corresponding fields, including email, first name, last name, and password.

After entering a password, the system checks that it meets the security rules. A valid password must be at least 8 characters long and include at least one uppercase letter, one number, and one special character (such as $%&_). If the password meets the criteria, the system shows a visual confirmation. Otherwise, it explains what is missing so the user can fix it.

The user then accepts the **terms and conditions** and selects **"Create account"** to continue.

![Create account](/images/cedi-crear-cuenta.png)

The system sends a confirmation code to the email address provided. The user enters the code in the field and clicks **"Verify code"**. If the code is correct, registration continues. If it is invalid, the user is notified and can request a new code.

![Verification code](/images/cedi-codigo.png)

After the account is verified, the user is taken to the sign-in screen and must enter the registered email and password. The system validates the credentials; if they are wrong, an error message is shown. If they are valid, the user can access the system.

![Sign in](/images/cedi-iniciar-sesion.png)
