---
title: Dashboard
---

The user accesses the **main Dashboard**, where the system displays a general summary of their account. In this view, the general balance, the total available balance, and a chart that reflects income and expenses according to the selected period are presented, allowing the user to analyze the behavior of their balance over time.

At the top, the system offers quick access to frequent actions, such as **New transfer**, **International transfer**, and **New dispersion**. By selecting any of these options, the user is automatically redirected to the corresponding flow; otherwise, they remain on the dashboard.

---

The system also presents a summary of the main sections. In the **Transfers** section, the user can view pending operations and, by selecting **"View all"**, is directed to the complete transfers section.

Similarly, in the **Dispersion** section, pending dispersions are displayed with the option to consult the complete list through **"View all"**. Likewise, in the **Invoices** section, the user can view pending invoices and access the complete billing section.

In the **Recent transactions** section, the system displays a list with the latest operations performed. If transactions exist, they are presented on screen; otherwise, the message **"No recent transactions"** is displayed.

---

The side menu allows navigation between the different system modules, including **Dashboard**, **Transfers**, **Dispersion**, **Invoices**, **Agents**, and **Support**. By selecting any of these options, the system redirects the user to the corresponding section.

In the lower left corner, the system displays the user profile (name and email), along with options to change the visual theme between light mode and dark mode, as well as the option to log out. It is important to note that, before performing any operation, the user must access their profile and activate two-factor authentication (2FA).
